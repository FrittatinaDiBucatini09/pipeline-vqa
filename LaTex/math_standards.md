**Context:**
I am writing a thesis on Medical Visual Question Answering (VQA) and Grounding. You are tasked with writing/completing a chapter. To ensure strict consistency with the previous chapters, you **MUST** use the following mathematical notation standard. Do not use different symbols for these concepts.

### 1. Input Spaces and Data
| Symbol | Description |
| :--- | :--- |
| $\mathcal{I}$ | Mathematical space of Images. |
| $\mathcal{Q}$ | Mathematical space of Textual Queries. |
| $\mathcal{A}$ | Mathematical space of Answers. |
| $\mathbf{I}$ | Input Image tensor ($\in \mathbb{R}^{H \times W \times C}$). |
| $q$ | Raw natural language query (free text). |
| $q'$ | Expanded or enriched clinical prompt (after NLP processing). |
| $a_{\text{gt}}$ | Ground-Truth Answer (reference). |
| $a^*$ | Predicted Answer generated by the model. |

### 2. Models and Encoders
| Symbol | Description |
| :--- | :--- |
| $f_{\text{vis}}$ | Visual Encoder function (ViT backbone within $f_{\text{VL}}$). |
| $f_{\text{text}}$ | Textual Encoder function (domain-specific biomedical text encoder within $f_{\text{VL}}$). |
| $f_{\text{VL}}$ | Contrastive Vision-Language encoder (dual-encoder backbone for cross-modal attention). |
| $f_{\text{seg}}$ | Segmentation foundation model for medical image segmentation. |
| $\mathcal{N}$ | Biomedical Named Entity Recognition (NER) module. |
| $\mathcal{G}$ | Generative Language Model used for query expansion. |
| $\theta_{\text{vis}}, \theta_{\text{text}}$ | Learnable parameters specific to the visual and textual encoders. |
| $\Theta$ | Global set of hyperparameters for the entire pipeline. |
| $\mathcal{M}_{\text{VQA}}$ | The downstream VQA Model (inference engine). |
| $\mathcal{S}_1, \dots, \mathcal{S}_4$ | The sequential macro-stages of the pipeline. |

### 3. Latent Representations (Embeddings)
| Symbol | Description |
| :--- | :--- |
| $\mathbf{v}$ | Continuous Visual Embedding vector. |

### 4. Grounding, Masks, and Segmentation
| Symbol | Description |

### 5. NLP and Query Routing
| Symbol | Description |
| :--- | :--- |
